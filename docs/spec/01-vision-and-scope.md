# 01. ビジョンとスコープ

## コンセプト

**Byond(仮称)** — 横16:9の紙芝居動画を作るための、個人専用フラットデザイン動画制作ツール。

Vyondの「視覚的編集の快適さ」と「紙芝居動画への最適化」を引き継ぎつつ、Vyondにない**キャラクターを0から自作できるクリエイター機能**を中核に据える。

## 基本方針: スモールスタートの設計判断

Vyond型デザインがスモールスタートに最適である理由を、そのまま設計原則にする:

1. **顔は1方向のみ作画** — キャラの向き変更はミラー反転(scaleX=-1)で表現。多方向の作画コストを排除
2. **等身の高いキャラクター**(5.5〜6.5頭身) — Vyond Business Friendly風。デフォルメより汎用的で、ビジネス系・解説系・コメディ系のどの紙芝居にも使える
3. **固定スケルトン** — リグはシステム既定(humanoid-v1)。ユーザーはリグを組まず、絵を描いてピンを打つだけ
4. **シーン相対タイム** — 動画はシーン(カット)の連結。タイミングは常にシーン内相対で、並べ替え・複製が壊れない(Vyondの編集快適性の本質)

## 4つのリファレンスから取る要素

| リファレンス | 取る要素 | 取らない要素 |
|---|---|---|
| **Vyond** | 視覚的編集(ドラッグ&ドロップ、右クリックメニュー、Replace機能)、プリセットアクション、enter/exit効果、シーンサムネ帯、アセット検索 | クラウド/共同編集、多方向キャラ、サブスク的アセット依存 |
| **Roblox** | 顔=自由なデカールという哲学。眉・目・口をユーザーが完全に自由デザインでき、組み合わせで表情が生まれる | 3D、ブロック体型 |
| **Live2D** | 髪の前髪・中間髪・後ろ髪の3層分離と振り子物理(揺れ幅・剛性・減衰のパラメータ設計) | メッシュ変形、フル微分パラメータ(重すぎる) |
| **Piotr Antkowiak (16personalities MBTIイラスト)** | アートディレクション: 角丸ジオメトリック、フラット塗り、どこか懐かしいRoblox的素朴さ、細長い手足 | (スタイル参照のみ) |

## v1スコープ

- キャラクタークリエイター(パーツ作画・顔シェイプセット・3層髪・パレット)
- プリセットリグアニメーション(idle/walk/run/talk/point等 約10種)
- 髪の物理(走ると躍動する)
- 表情システム(眉×目×口の組み合わせプリセット)
- シーンエディタ(背景・小道具・テキスト・enter/exit・カメラ)
- 音声(VO/BGM)+ 簡易リップフラップ
- MP4書き出し(1080p)

## 非ゴール(v1ではやらない)

- 多方向の顔・体(1方向+ミラーのみ)
- カスタムポーズエディタ(キーフレーム手付け) → Phase 7
- 母音リップシンク(あいうえお) → Phase 6+
- 3D、メッシュ変形、ボーンIKエディタ
- クラウド保存・共同編集・モバイル・配布(完全ローカル・自分専用)
- 多言語UI(日本語のみ)

## 成功条件

- **3分の紙芝居動画を1日で作れる**(キャラ既存・台本ありの状態から)
- **新キャラを1〜2時間で作れる**(テンプレ改変なら30分)
- 走るシーンで髪が「気持ちよく」なびく(主観評価だが最重要品質)

## アートスタイルガイド(スターターアセット制作指針)

- **形状**: 角丸ジオメトリック。シルエット明瞭。ベジェより基本図形の組み合わせ優先
- **線**: 基本なし(塗り分けのみ)。必要時のみ同系色の暗い細線
- **陰影**: 1段階暗い色のワンポイント(任意)。グラデーション禁止
- **体型**: 5.5〜6.5頭身、手足やや細長(Antkowiak風)、手はミトン+親指
- **顔テンプレ**: 点目+シンプル眉+線の口(ただしユーザーは完全自由に変更可能)
- **パレット**: キャラ毎に6スロット(skin / hair / primary / secondary / accent / line)
- **背景**: キャラより彩度・情報量を抑える(紙芝居の主役はキャラと文字)
