# 05. アニメーションランタイム

## 基本構造

- **クリップ** = humanoid-v1の各ボーンへのキーフレームトラックの束。全キャラ共有(同一スケルトンのため、回転キーがそのまま流用できる)
- **表情はクリップから独立**。ボディアニメと表情は直交するレーン(「走りながら笑う」「歩きながら怒る」が自由に組める)。これがVyondの "Action × Expression" の快適さの再現
- ハンドシェイプ・z-overrideはクリップが指定できる(walkは`relax`、pointは`point`等)

## クリップフォーマット

```jsonc
{
  "formatVersion": 1,
  "id": "run",
  "label": "走り",
  "duration": 0.52,           // 秒(fpsは持たない。時刻は全て秒)
  "loop": true,
  "virtualVelocity": 580,     // その場再生時の仮想移動速度 [u/s](髪物理・moveTo同期用)
  "tracks": {
    "bones": {
      "thighL": { "rot": [[0, 48, "sineInOut"], [0.26, -44, "sineInOut"], [0.52, 48]] },
      "torso":  { "rot": [[0, 13]] }            // 前傾(定数キー)
    },
    "root": { "y": [[0, -2, "quadOut"], [0.13, -14, "quadIn"], [0.26, -2]] },
    "handShape": [[0, "fist"]]
  }
  // zOverrides / events は Phase 4+ で追加
}
```

- キー = `[時刻, 値, イージング?]`。イージングは**そのキーから次のキーまでの区間**に適用。セット: `linear / sineInOut / quadIn / quadOut / quadInOut / backOut / bounceOut`(既定 linear)
- **ループ規約**: 暗黙のwrap補間はしない。末尾キーを先頭値・`t=duration` で明示的に閉じる(テストで機械検証)
- ルートの平行移動はY(バウンス)のみクリップに持つ。**X移動はシーン側の移動指示**(enter効果や移動アクション)が持ち、クリップは「その場で走る」。歩行速度と足の周期の同期は `virtualVelocity` を基準にシーン側が再生速度を連動させて解決

## v1プリセットクリップ(10本)

| id | 内容 |
|---|---|
| idle | 呼吸(torso/headの微揺れ)。デフォルト |
| walk | 歩行ループ。腕振り付き |
| run | 走行ループ。前傾・バウンス強(髪物理の見せ場) |
| talk1 | 説明ジェスチャA(片手を開いて差し出す) |
| talk2 | 説明ジェスチャB(両手・相づち) |
| point | 指差し(handShape=point)。方向は腕角度パラメータ |
| wave | 手を振る |
| nod | うなずき |
| headShake | 首を横に振る(=ミラー反転の連続ではなくhead回転) |
| jump | その場ジャンプ(髪・物理の見せ場) |

各クリップは `speed`(再生倍率)・`loop` をシーン側から指定可能。

## 合成パイプライン(評価順)

```
1. ベースクリップFK評価(補間) → ボーンローカル回転
2. 表情レーン評価            → 顔シェイプ選択+オフセット
3. ワールドトランスフォーム解決(ボーン階層)
4. 髪・揺れ物理(ワールド速度入力, 固定dt)
5. ミラー反転(キャラの向き)
6. FrameState出力 → render層へ
```

- クリップ間遷移: **0.22秒の回転lerpクロスフェード**(smoothstep、デフォルト、指定可)。片方にしか無いボーンはレスト(0°)とブレンド
- 物理は遷移中も連続(切れない)
- ミラー反転(向き)は**表示レイヤのscaleX=-1**で実装(ポーズ数学は無加工)。仮想速度の符号は呼び出し側が反転する

## トーク(会話モード)

Vyondの「セリフを割り当てると喋る」体験の再現:

- シーン側で `talk: { audioRef, clip: "talk1", expression: "smile" }` を指定
- 音声の長さに合わせてクリップをループ、リップフラップ(04参照)を自動駆動
- 音声なしトーク(尺だけ指定)も可
