# 03. キャラクターシステム(支柱機能)

本ツールの中核。**「固定スケルトン × ピン留めパーツ = オートリグ」**という一点に全てを賭ける設計。

## 設計原則

1. **ユーザーにリグを組ませない**。骨格はシステム既定の `humanoid-v1` 固定。ユーザーは「絵を描いてピン(関節位置)を打つ」だけで、骨長はピン間距離から自動算出される
2. **1方向作画**。向き変更はミラー反転。非対称な髪型・服も反転を許容(フラットデザインの記号性で吸収)
3. **プリセットアニメは全キャラ共有**。同一スケルトンなので回転キーがそのまま流用できる(リターゲット不要)。体格差はボーン長の違いとして自然に吸収される
4. **色は間接参照**。パーツはパレットスロットを参照し、キャラ全体の配色変更が一発で済む

## スケルトン定義: humanoid-v1

| bone | parent | 備考 |
|---|---|---|
| root | — | 腰。移動・走りのバウンスはここ |
| torso | root | 胴。shoulderL/R・hipL/R・neckTopアンカーを内包 |
| head | torso (neckTopに接続) | 首の表現はhead回転で代替(neckボーンなし) |
| upperArmL/R | torso (shoulderに接続) | |
| forearmL/R | upperArm | |
| handL/R | forearm | シェイプセット差し替え式(後述) |
| thighL/R | torso (hipに接続) | |
| shinL/R | thigh | |
| footL/R | shin | |

計15ボーン。各ボーンは回転1自由度(rootのみ+平行移動)。IKなし(v1)。

## パーツスロットとデフォルトZ順

奥(描画順先)→ 手前。Lを手前側(カメラ側)と規約する:

| z | スロット | 従属ボーン |
|---|---|---|
| 10 | hairBack | head |
| 20-22 | upperArmR / forearmR / handR(奥腕) | 各ボーン |
| 30-32 | thighR / shinR / footR(奥脚) | 各ボーン |
| 40-42 | thighL / shinL / footL(手前脚) | 各ボーン |
| 50 | torso | torso |
| 60 | head | head |
| 70-78 | 顔パーツ(→ 04参照) | head |
| 80 | hairMid | head |
| 85 | hairFront | head |
| 90-92 | upperArmL / forearmL / handL(手前腕) | 各ボーン |
| 95+ | アクセサリ(帽子・メガネ等は任意z) | head等 |

- z値は**パーツ毎に手動調整可能**(デフォルトを提供するだけ)
- クリップ・シーン側から **z-override** 可能(腕組みで腕を顔の前に出す等)

## ピン仕様

パーツ作画時にユーザーが打つアンカー。テンプレートのゴースト下絵が推奨位置を示す。

| パーツ種 | 必須ピン | 任意ピン |
|---|---|---|
| 四肢(upperArm/forearm/thigh/shin) | origin(親関節)・joint(子関節) | — |
| torso | origin(腰) | neckTop・shoulderL/R・hipL/R(未設定ならテンプレ比率で自動) |
| head | origin(首接続点) | — |
| hand | origin(手首) | **grip(小道具持たせアンカー)** |
| foot | origin(足首) | sole(接地点) |
| 髪ストランド | origin(根元) | — |

- ボーンの向き = origin→joint ベクトル。骨長 = ピン間距離
- バリデーション: 必須スロット欠落 / ピン未設定 / 骨長がテンプレ比±50%超(警告のみ)

## パートエディタ(作画機能)

「0から作れる」を、フルベクタツールではなく**プリミティブ合成**で実現する。対象スタイル(角丸ジオメトリック)はプリミティブで描くのが最速で、絵柄の統一感も構造的に担保される。

- **シェイプツール**: 矩形 / 角丸矩形 / 楕円 / カプセル / 多角形 / ベジェパス(上級用)
- **塗り**: パレットスロット参照 or 固定色。ストロークは任意(同系暗色推奨)
- **編集**: 頂点編集・複製・グループ・前後入替・整列
- **ミラー複製**: 左腕を描いたら右腕へ自動反転コピー(ピンも反転)
- **SVGインポート**: 1パーツ=1ファイル、path/基本図形のみサポート(Illustrator等で描く人向けの逃げ道)
- **ガイド**: テンプレ標準体型のゴースト表示+ピン推奨位置マーカー+頭身グリッド
- **プレビュー**: 編集中パーツを装着した全身を常時右側に表示し、ポーズ(idle/walk)を回して確認できる

## ハンドシェイプセット

手は口と同じ「差し替え式」。クリップが `handShape` を指定する。

- v1必須5種: `open` / `fist` / `point` / `hold` / `relax`
- 手はL側(手前)で1回だけ作成し、R側は実行時に自動ミラーされる
- テンプレートがデフォルトセットを提供。ユーザーは差し替え・追加可能
- `hold` + gripピンで小道具(コーヒー、スマホ等)を持たせる

## パレットスロット

`skin / hair / primary / secondary / accent / line` の6スロット+自由追加。キャラのプロパティパネルで一括変更 → 全パーツに波及。肌色と髪色の整合(眉の色=hair等)もスロット参照で自然に解決。

## 作成フロー

1. **テンプレ改変**(推奨): スターターキャラ(男女2体)を複製 → パーツを差し替え/再作画
2. **空から**: スロットリストを順に埋める(ゴースト下絵あり)
3. 完成時に**コンタクトシート出力**(ポーズ4種×表情6種のグリッドPNG)で品質確認

## プロポーション

v1ではパーツの作画サイズがそのまま体格になる(描いたまま)。頭身スライダ等のパラメトリック変形はPhase 7(パーツ再加工が不要になる仕組みが必要なため)。
