# 06. シーンエディタ(Vyond的編集体験)

Vyondの本質的な強み =「視覚的編集の快適さ」を仕様として明文化し、継承する。

## 画面構成(Vyond Studio準拠)

```
┌──────┬──────────────────────────┬─────┐
│ アセット │        ステージ(キャンバス)        │ プロパティ│
│ パネル  │   1920×1080 / グリッド / スナップ   │ パネル  │
│ (検索/  │                                  │ (選択中 │
│ タブ)   │                                  │ 要素)   │
├──────┴──────────────────────────┴─────┤
│ シーンサムネ帯(並べ替え/複製/追加)                      │
│ 選択シーンのタイムライン(要素レーン)                     │
└─────────────────────────────────────────┘
```

## シーン相対タイム(編集快適性の核)

- プロジェクト = **シーン(カット)の配列**。グローバルタイムラインは存在しない
- 全タイミングはシーン内相対(t=0がシーン開始)。**シーンの並べ替え・複製・削除が他シーンに一切影響しない**
- シーン長 = 手動指定 or「音声にフィット」or「最長要素にフィット」
- シーン間トランジション: cut / fade / wipe / slide(v1はこの4種)

## 要素モデル

シーンに置けるもの: キャラ / 小道具(SVG/PNG) / 背景 / テキスト / 吹き出し / 音声。

```jsonc
{
  "ref": "characters/hana.byc.json",
  "transform": { "x": 480, "y": 700, "scale": 0.9, "flipX": false },
  "z": 3,
  "enter": { "type": "slideL", "delay": 0.0, "dur": 0.4 },
  "exit":  { "type": "fade", "at": "sceneEnd-0.3", "dur": 0.3 },
  "actions":     [{ "t": 0.5, "clip": "walk", "moveTo": { "x": 960 }, "speed": 1.0 }],
  "expressions": [{ "t": 0.0, "preset": "neutral" }, { "t": 1.2, "preset": "surprised" }],
  "gaze":        [{ "t": 1.2, "x": -0.6, "y": 0 }],
  "talk":        { "t": 2.0, "audioRef": "assets/audio/vo_03.wav", "clip": "talk1" }
}
```

- enter/exit効果: cut / fade / slide(4方向) / pop(scale bounce)。Vyondと同じ「要素の出入りはエフェクトレーン」という考え方
- `moveTo` 付きアクションで歩行・走行の横移動(クリップ再生速度と移動速度を同期)

## ステージ直接操作

- ドラッグ移動 / ハンドルで拡縮回転 / ダブルクリックでアクション設定ダイアログ
- **スナップ**: 3分割グリッド・中央線・他要素エッジ・セーフエリア表示
- **右クリックメニュー**(Vyond準拠): コピー / ペースト / 複製 / 反転 / 順序(最前面へ等4種) / 整列 / 分布 / **ロック(Ctrl+L)** / **Replace**
- 複数選択・グループ化

## Replace(Vyond最強機能の再現)

選択要素を別アセットに差し替え、**transform・タイミング・アクション・表情を全て維持**する。
- キャラ→キャラ: アクション/表情/トークがそのまま引き継がれる(同一スケルトンなので完全互換)
- 小道具→小道具: transformのみ維持
- 「キャラを差し替えて同じ動画をもう1本」が数クリックで済む。紙芝居量産の要

## タイムライン(シーン内)

- 要素毎に1レーン: enterブロック / アクションブロック / exitブロック / 表情マーカー / トークブロック
- ブロックのドラッグで時刻調整、端ドラッグで長さ調整
- 音声波形表示(VOレーン)
- 再生ヘッド+スペースキーで再生/停止。シーン単独再生と通し再生

## カメラ

- シーン毎にカメラボックス(x, y, zoom)のキーフレーム。ease付きパン/ズーム
- プレビュー・書き出しで同一評価(02の評価器原則)

## 音声

- **VO**: シーン要素として配置(波形表示・オフセット調整・トーク連動)
- **BGM**: プロジェクトレベルで配置し複数シーンを跨げる(唯一のシーン横断要素。シーン開始基準のアンカーで保持)
- 簡易ボリューム(要素毎gain)。VO中BGM自動ダッキング(-6dB)はPhase 6
- ブラウザ内マイク録音(MediaRecorder)はPhase 6検討

## テキスト・吹き出し

- テキスト: ローカルフォント選択(queryLocalFonts)・サイズ・色・縁取り(紙芝居の字幕に必須)・行間
- 吹き出し: 角丸/雲/トゲ(叫び)の3種+しっぽ位置ドラッグ。中にテキスト内包
- enter/exit効果はテキストにも適用可(タイプライター表示はPhase 6)

## アセットライブラリ

- タブ: キャラ / シーン(背景) / 小道具 / テキスト / 音声(Vyondのタブ構成踏襲)
- タグ検索(日本語)・最近使った・お気に入り
- 取り込み: SVG / PNG / WAV / MP3をプロジェクトフォルダへコピー登録
- スターターパック: 背景10種・小道具30種・吹き出し3種(制作計画は 08 ロードマップ参照)

## 書き出し

- WebCodecs: H.264 + AAC → MP4(mp4-muxer)。1080p/720p、30/24fps
- 評価器をフレーム順に回す(プレビューと同一コード)。プログレスバー+中断可能
- フォールバック: PNG連番+WAV書き出し(ffmpegで結合する手順をドキュメント化)
- 静止画書き出し: 現在フレームをPNG(サムネ用)。コンタクトシート出力もここに統合

## ショートカット(v1)

| キー | 動作 |
|---|---|
| Space | 再生/停止 |
| Ctrl+C/V/D | コピー/ペースト/複製 |
| Ctrl+L | ロック |
| Ctrl+Z / Shift+Ctrl+Z | undo / redo |
| 矢印 / Shift+矢印 | 1px / 10px ナッジ |
| Delete | 削除 |
| [ / ] | 背面へ / 前面へ |
